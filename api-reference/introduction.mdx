## API Overview

OrderPath provides a transaction routing service that submits your transactions to MEV builders for optimal execution and potential refunds. The API uses standard JSON-RPC 2.0 protocol, making it compatible with any Ethereum RPC client.

## Endpoint

**Production:** `https://rpc.orderpath.xyz`

OrderPath uses standard JSON-RPC 2.0, so you can use it as a drop-in replacement for any Ethereum RPC endpoint.

## Supported Methods

OrderPath supports all standard Ethereum JSON-RPC methods. Unsupported methods are automatically forwarded to an upstream RPC provider.

**OrderPath-specific methods:**

- `eth_sendRawTransaction` - Submit transaction to MEV builder
